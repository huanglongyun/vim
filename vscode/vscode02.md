day20

目标：操作多个vscode窗口

1. 打开新的vscode窗口 shift + ctrl + n

2. 选择工作区 ctrl + r
    ```
        // settings.json
            "vim.handleKeys": {
                "<C-r>": false,
            },
    ```
3. 多个窗口切换
    方式一 安装插件easy windows switcher
    方式二 键盘快捷方式 搜 切换窗口 自己设置 alt e

4. 关闭vscode窗口 shift ctrl w
